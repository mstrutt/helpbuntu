What is HelpBuntu? - HelpBuntu is a support site, created and administered by Michael Strutt, for Ubuntu - the Linux operating system. The site contains Frequently Asked Questions for common problems, along with tutorials and useful reference guides. A piece I am particularly proud of is the Metapixel Generating Engine for creating photo-mosaics. If nothing else on the site solves your problem then you can send in your problem via the contact page and get support from the HelpBuntu team.

So what is Ubuntu? - Ubuntu is an operating system, like the more commonly known Mac of Windows, however unlike these it is based on Linux. Linux is a free disk operating system (the main set of commands that control the computer) and Ubuntu is one of the many versions of desktop environments written for Linux. It is free with good community support (and support sites such as this one) and updates are regularly provided to download at your leisure. To see how to install it, see my installation page.

What about Software? - Most commercially sold software will not work on Ubuntu (though there are programs such as WINE that might) but they are not needed. There is a vast supply of free software to cater for every need. A directory of all the main software available software to install is available from the main menu and lots of outsider applications also freely distributed. I have compiled a list of some of the best on my applications page.

What will this help cost me - Absolutely nothing. Helpbuntu upholds the Ubuntu belief that all this should be free. However if you feel you would like to help with the running costs of 30GBP per year then please spare what you can by clicking on the donate button at the top of any page.
