CSS Style Sheets

A CSS Style sheet is linked to by placing a link in the within the <head> </head> tags of a web page. The style sheet is an external document with a ".css" extension containing css styles. I am assuming that the file is called style.css and is in the same folder as the file using its style. It is linked to with the following link:

This is the simplest way to style multiple pages in the same way as it only requires one link at the top of the page and then you only have to write the style once. Also if you wish to change the style then it only has to be change on the style sheet and all the relevant styles are changed.
